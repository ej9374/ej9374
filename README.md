## 👋 안녕하세요, 백엔드 개발자 **김은지**입니다.

### "서비스의 빈틈을 기술적 밀도로 채우는 백엔드 개발자입니다."

저는 **Java**와 **Spring Framework**를 기반으로 **안정적인 아키텍처**와 **높은 코드 품질**을 최우선으로 합니다.
새로운 기술 도입을 두려워하지 않으며, 팀의 개발 속도를 가속화하는 **문서 기반 리더십**을 발휘합니다.

### ✨ My Core Values

| 가치 (Value) | 핵심 내용 (Key Focus) | 기술적 실천 (Technical Practice) |
| :--- | :--- | :--- |
| **💡 Learning Agility** | 서비스에 필요한 기술은 끝까지 파고들어 내 것으로 만듭니다. | 단순 구현을 넘어, **STOMP 채팅 보안 리팩토링** 및 **QueryDSL 동적 쿼리**로 최적화를 고민합니다. |
| **🤝 Leadership** | 팀의 속도를 높이는 '문서 기반 소통'을 지향합니다. | **API 명세서 정의**와 Notion 로드맵 관리를 주도하여 협업 효율을 극대화합니다. |
| **💎 Code Quality** | 유지보수의 핵심은 '명확한 규칙'과 '통일성'에 있습니다. | **Global Exception Handler** 도입으로 에러 처리를 표준화하고, **API 공통 응답 규격**을 설계했습니다. |
